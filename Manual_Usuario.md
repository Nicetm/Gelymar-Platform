# Manual de usabilidad plataforma Gelymar

## 1. Acceso general
- Usa la URL que corresponde a tu rol y tus credenciales (correo/RUT y contraseña).
  - Administrador: `http://172.20.10.151:2121`
  - Vendedor: `http://172.20.10.151:2123`
  - Cliente: `https://logistic.gelymar.cl`
- Login: escribe usuario y contraseña y pulsa “Ingresar”. Si el sistema te pide iniciar sesión nuevamente, repite el proceso.
- Modo claro/oscuro: en la barra superior hay un interruptor para cambiar el tema cuando lo necesites.

## 2. Administrador
- Órdenes: lista de órdenes con búsqueda y filtros. Al hacer clic abres sus documentos.
- Clientes: buscador y tabla de clientes; desde la columna de acciones abres las carpetas del cliente y sus órdenes.
- Vendedores: listado de vendedores con datos básicos y estado en línea/desconectado.
- Documentos: dentro de cada carpeta u orden puedes:
  - Buscar, ver y descargar documentos.
  - Marcar si el documento es visible para el cliente.
  - (Solo admin) Subir archivos y generar archivos por defecto cuando falten.
- Configuración:
  - Lenguaje: selecciona el idioma disponible para la interfaz.
  - Chat: acceso al chat interno para comunicarte con otros usuarios.

## 3. Vendedores
- Órdenes: consulta las órdenes asociadas a tus clientes y abre sus documentos.
- Clientes: busca clientes y abre sus carpetas; solo verás acciones de consulta.
- Documentos: desde las carpetas puedes buscar y descargar documentos. Las opciones de administración (subir, generar por defecto, editar, eliminar) no aparecen para vendedores.

## 4. Clientes
- Documentos: en el “Document Center” puedes ver, filtrar y descargar los documentos de tus órdenes. Al abrir un archivo su estado cambia a “Viewed”.
- Chat: disponible cuando la interfaz lo muestre para comunicarte con el equipo de Gelymar.

## 5. Consejos rápidos
- Si un botón no aparece, verifica que estés en la URL correcta para tu rol.
- Si la descarga de un documento falla, inicia sesión de nuevo y reintenta.
- Usa la búsqueda y los filtros para encontrar más rápido órdenes o documentos.
